# :foggy:如何使用：

1. 先填写`data.conf`，参数说明（参数填写不需要加引号）：
  
  | 参数  | 说明  |
  | --- | --- |
  | `phone` | 校园网登录的手机号码，如`xxxxxxxxxxx` |
  | `password` | 校园网登录密码 |
  | `operator` | 运营商名称，如：中国电信 |
  

2. 须确保您的`data.conf`、`nuist-auto-login.py`或`nuist-auto-login.exe`在同一目录下。

3. 如果您在使用中遇到任何问题，您有两种途径可以反馈：
